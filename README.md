# Installation

    mkdir -p ~/.oh-my-zsh/custom/themes
    cd  ~/.oh-my-zsh/custom/themes
    git clone https://github.com/oliverheins/oh-my-zsh-clean-oh-theme.git clean-oh-theme
    cp clean-oh-theme/clean-oh.zsh-theme .

In your ~/.zshrc file, modify the line with `ZSH_THEME` to `ZSH_THEME="clean-oh"`.
